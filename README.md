# Pharmaceutical-Financial-Performance-Dashboard

📌 Overview

This project focuses on analyzing pharmaceutical sales data to evaluate revenue trends, product performance, and demand patterns. The objective is to derive business insights that support financial analysis and decision-making, similar to FP&A functions.

🎯 Objectives
Analyze monthly revenue trends across pharmaceutical products
Identify top-performing drug categories
Evaluate sales variability and demand patterns
Build an interactive dashboard for financial performance tracking
🛠️ Tools Used
Power BI (Data Transformation & Visualization)
Excel (Data Cleaning & Preparation)
📂 Dataset Description

The dataset contains monthly pharmaceutical sales data across multiple drug categories, including:

Pain Relief
Fever Medication
Respiratory Drugs
Allergy Medication
Anxiety Drugs
Sedatives

Each row represents monthly sales performance across categories.

🔧 Data Preparation
Cleaned dataset by removing anomalies and ensuring consistent date formats
Created a Total Sales column for overall revenue tracking
Transformed data using Power BI (Power Query)
Applied unpivoting to convert wide-format data into analysis-friendly structure
📊 Dashboard Features
KPI Cards: Total Revenue, Average Monthly Revenue, Peak Revenue
Monthly Revenue Trend: Line chart showing performance over time
Category-wise Analysis: Revenue contribution by drug category
Interactive Filter: Drug category slicer for dynamic insights
💡 Key Insights
Revenue shows fluctuations across months, indicating variability in demand
A few drug categories contribute a significant share of total revenue
Seasonal demand patterns are visible in respiratory and allergy-related products
Overall performance remains stable with periodic spikes
📈 Business Impact

This analysis helps in:

Understanding key revenue drivers
Supporting financial planning and performance tracking
Identifying demand patterns for better decision-making
🚀 Conclusion

The project demonstrates the use of Power BI for transforming raw data into meaningful financial insights. It reflects how data-driven analysis can support FP&A functions such as performance evaluation and business decision-making.
